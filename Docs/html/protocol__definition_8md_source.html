00015 All messages send or received should have a register and a

00021  - Data[0]
00022  - ...
00023  - Data[size - 1]

00025 If you have a message with only a register needed you have to set size to 0.
00026 The message transmitted will be :
00027 - Slave addr

00029 - 0
